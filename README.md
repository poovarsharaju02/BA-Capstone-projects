# BA-Capstone-projects
 📊 Business Analytics Capstone Project

 Enhancing Operational Efficiency in a Multi-Specialty Hospital

 Overview

This project is a Business Analytics capstone focused on Enhancing Operational Efficiency in a Multi-Specialty Hospital.
The objective is to derive insights and support decision-making using data-driven approaches.



🎯 Objectives

Reduce patient wait times to improve overall service efficiency
Eliminate appointment scheduling conflicts and double bookings
Optimize utilization of medical staff and resources
Improve communication and coordination between department
Implement data-driven decision-making through dashboards and analytics
enhance patient satisfaction and service quality
ensure smooth adoption of digital solutions across the organization

🗂️ Project Contents

* BRD document
*Stakeholder engagement 
* Risk matrix
*data analysis  
* Presentation – Summary of key insights
* Supporting Files – Charts, datasets, visuals
